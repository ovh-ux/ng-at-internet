atInternetProvider

Description

Provider allowing configuration for atInternet service.

Methods

  • setDebug(state)

    Enable or disable logging of tracking data in the Javascript console.

    Parameters
    ParamTypeDetails
    stateBoolean

    True to enable console logs, false otherwise.

  • setDefaults(def)

    Configure default data to be sent with each tracking data.

    Parameters
    ParamTypeDetails
    defObject

    Default values to be sent.

  • setEnabled(state)

    Enable or disable tracking.

    Parameters
    ParamTypeDetails
    stateBoolean

    True to enable tracking, false otherwise.
